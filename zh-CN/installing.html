安装

安装该软件包（或将其添加到你的 requirements.txt 文件）。

$ pip install sphinx_rtd_theme

在你的 conf.py 文件中：

import sphinx_rtd_theme

extensions = [
    ...
    'sphinx_rtd_theme',
]

html_theme = "sphinx_rtd_theme"

参见

支持的浏览器

官方支持和测试的浏览器/操作系统组合。

支持的依赖

官方支持的 Python、Sphinx 和其他依赖的版本。

注解

把这个主题作为一个插件添加到你的翻译输出中，才能实现主题字符串的本地化。如果这些字符串在你的输出中没有被翻译出来，要么是我们缺乏适合你所在地区的本地化字符串，要么是你使用的是一个旧版本的主题。

通过 Git 或下载

警告

直接从版本库源码安装的方式已经过时，不推荐。从 release 3.0.0 版本开始，静态 assets 将不包括在版本库中。

将 sphinx_rtd_theme/sphinx_rtd_theme 储存库的链接或子树放到你文档的 docs/_themes/sphinx_rtd_theme 中，然后在你的 Sphinx conf.py 文件中添加以下两个设置：

html_theme = "sphinx_rtd_theme"
html_theme_path = ["_themes", ]
